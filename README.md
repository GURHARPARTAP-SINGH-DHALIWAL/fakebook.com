# FakeBook.com

A Social Media WebApp.


## Technologies Used

* React.Js

* Redux

* Thunk

* React-Redux

* Socket.io-Client

## Specs.

*  Login/Register Using JWT Tokens.

* User Profile View.

* Edit User Credentials.

* Posting.

* Liking and Commenting on a Post.

* Add and Remove Firends.

* Search Users.

* Chatting with other Users.
